# Аналитика продаж — Интерактивный дашборд

Современный адаптивный дашборд для анализа продаж на React + TypeScript.  
Данные загружаются из Excel-файла. Архитектура позволяет подключить CRM без изменения компонентов.

**Деплой:** https://larisamashkova.github.io/sales-dashboard/

## Технологии

- React 19 + TypeScript
- Vite — сборка
- Tailwind CSS — стилизация
- Recharts — графики
- Zustand — управление состоянием
- SheetJS (xlsx) — импорт/экспорт Excel

## Запуск

```bash
npm install
npm run dev
```

Открой `http://localhost:3000`.

## Сборка и деплой на GitHub Pages

```bash
npm run build
cd dist
git init
git add .
git commit -m "deploy"
git remote add origin https://github.com/larisamashkova/sales-dashboard.git
git push -f origin main:gh-pages
```

Файл `Sample_Sales_Data_MVP_Extended.xlsx` должен лежать в корне репозитория (рядом с `index.html` в ветке `gh-pages`), чтобы дашборд подтягивал его автоматически.

## KPI (ключевые метрики)

| Метрика | Описание |
|---------|----------|
| Общий объём продаж | Сумма выручки за выбранный период |
| Количество транзакций | Число заказов |
| Продано единиц | Суммарное количество проданных товаров |
| Средний чек | Выручка / количество транзакций |
| Доля рынка | Доля выбранного сегмента в общих продажах |
| Темп роста | Динамика относительно предыдущего периода |

## Фильтры

- **Период** — начальная и конечная дата (date picker)
- **Регион** — выбор региона (5 регионов)
- **Категория** — выбор категории товаров
- **Менеджер** — выбор продавца/менеджера
- **Поиск** — текстовый поиск по товару, категории, региону

## Графики

1. **Динамика продаж** — линейный график (продажи + прибыль по месяцам)
2. **Продажи по регионам** — столбчатая диаграмма (все регионы, игнорирует фильтр региона)
3. **Распределение по категориям** — кольцевая диаграмма с долями в % (все категории, игнорирует фильтр категории)

## Аналитическое резюме

Блок «Автоматические выводы» формирует управленческие наблюдения:
- Общая выручка, прибыль и рентабельность
- Регион-лидер и отстающий регион
- Категория-лидер с долей в %
- Динамика продаж месяц к месяцу
- Лучший менеджер по объёму продаж
- Сравнение среднего чека с общим показателем

## Структура страницы

1. **Загрузчик файла** — выбор Excel или drag & drop (с индикацией)
2. **Фильтры** — период, регион, категория, менеджер, поиск
3. **KPI-карточки** — 6 ключевых метрик с анимациями
4. **Динамика продаж** — линейный график
5. **Продажи по регионам** — столбчатая диаграмма
6. **Распределение по категориям** — кольцевая диаграмма
7. **Автоматические выводы** — аналитическое резюме
8. **Экспорт** — выгрузка отфильтрованных данных в Excel

## Архитектура

```
src/
├── api/apiService.ts      # Адаптер для будущего API/CRM
├── components/             # UI-компоненты
├── store/useStore.ts       # Zustand-хранилище
├── types/index.ts          # TypeScript-типы
└── utils/
    ├── calculations.ts     # Вычисления KPI, фильтры
    └── dataParser.ts       # Парсер Excel с валидацией
```

Для перехода с Excel на CRM достаточно реализовать методы интерфейса `DataSource` в `apiService.ts` — компоненты не требуют изменений.
